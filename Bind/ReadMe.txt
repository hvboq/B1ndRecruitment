 프로그램 제목 : Dayary , 하루 일기라는 의미로 Day, Diary 두 단어를 합성 하였습니다.
 
 기초 설계 : 개발 기간이 촉박하므로 기존의 윈폼 개인 프로젝트였던 일기장의 기능을 변경하여, 개발 하기로 하였습니다.
 컨트롤 10개 이상을 사용하여야 하여 간소하게나마 로그인, 회원가입 기능을 추가하였습니다.
 
 대표 기능 : 계정 관련 기능, 일기를 바탕화면에 저장 하는 기능, 일기를 표 형식으로 출력 하는 기능이 있습니다.
 
 스스로 돌아보기 : 파일 입출력을 사용하여 log와 Daysaves 를 종료할때 저장하고 다음에 켜지면 불러오는 기능이 필요합니다.
 일기 보기 기능에서 삭제와 수정 기능이 필요합니다.
 설정 관련 기능을 통하여 저장 경로나 날짜 형식등을 바꾸는 기능이 필요합니다.
 
 사용가능 환경 : 운영체제 윈10 이상, 폰트로 나눔고딕이 필요함. 혹여나 프로젝트 디버깅 에서 오류가 날 시 bin 폴더에 release의 exe 파일로도 실행이 가능합니다.
